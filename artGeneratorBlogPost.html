The JavaScript Art Generator

The JavaScript art generator was one of the first projects that I made on my own and pushed to GitHub when I began learning how to program. Despite its simplicity and my lack of experience at the time, I still believe that it is one of the more interesting and novel things I have created. I had originally set out to create a pong type game in JavaScript on an HTML canvas, but had accidentally forgotten to redraw the canvas background color on every frame after moving the cube. As a result the cube that was supposed to represent the pong ball ended up leaving a trail behind it. After realizing the potential of this cube to generate interesting patterns I decided to completely abandon my original idea.

As part of the first JavaScript homework for my class on building advanced web apps we had the option of creating any client based JavaScript application of our choice and adding it to our portfolio website. I decided to take the almost two year old code that had been sitting on github and add new features to it. The three new features that were added were the ability to pause the motion of the cubes, save the canvas as an image, and customize the colors of the boxes and background. I also rewrote the code so that it did a better job of following the DRY principles. Despite the new version having more features, it has considerably fewer lines of code.
