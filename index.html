Welcome to my website

My name is Clifford Wilson and I am in my final semester studying Computer Science at the University of New Orleans. I am also a student intern at the United States Naval Research Laboratory in Mississippi. This website aims to showcase some of my personal projects that I have worked on along with blog posts on various subjects that I find interesting.

Projects

U-Fit

Skills Used: Java, XML, SQLite, Android

U-Fit is an Android app which allows users to create custom workouts, perform them, and track their progress. U-Fit uses an SQlite database to keep track of user progress and created exercises.

Android Polling App

Skills Used: Java, XML, Android

As part of my coursework for my mobile development course at UNO I created an Android polling app with a partner. The app allows teachers to create templates for math problems which are then used to procedurally generate problems for students to solve.

JUCE Audio Effects VST

Skills Used: C++, JUCE

Audio programming is one of the most interesting topics in Computer Science to me and from my very first encounter with music production software I have wanted to build some of my own. C++ seems to be the standard language used in modern audio applications and it is considerably more complex than Java which I have used throughout my work in College. This project was a way for me to become acquianted with C++ before hopefully moving on to more complex topics such as building a wavetable synthesizer.

JavaScript Art Generator (Click To Demo)

Skills Used: JavaScript, CSS, HTML

This is a simple application I created to introduce myself to JavaScript. It draws artistic patterns on an HTML canvas in the form of bouncing cubes that leave behind a trail.

Phaser Asteroid Dodging Game(Click to Demo)

Skills Used: JavaScript(Phaser)

As part of my Advanced Web Apps course I was required to build a client-side browser app that rendered graphics and was actively controlled by the user. I decided that a basic 2D game in Phaser would satisfy those requirements. In this game you control a spaceship that needs to dodge asteroids as it flies through space.
